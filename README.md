# RasBot 🤖

RasBot is a ROS 2 mobile robot built using a Raspberry Pi 4, ESP32, and Slamtec RPLIDAR C1. The goal of this project is to learn and implement robotics concepts such as SLAM, autonomous navigation, and sensor integration.

## Hardware

- Raspberry Pi 4 (8GB)
- ESP32
- Slamtec RPLIDAR C1
- Robot chassis
- DC motors
- Motor driver
- Battery pack

## Software

- Ubuntu 24.04
- ROS 2 Jazzy
- SLAM Toolbox
- Navigation2 (Nav2)
- RViz2
